# AI Dropshipping Product Research Agent

An AI-powered SaaS platform that helps dropshippers identify profitable, low-competition products by combining automated data collection, quantitative scoring, and large language model reasoning.

## Core Features (MVP)
- Automated product discovery from e-commerce marketplaces
- Product viability scoring based on demand, competition, pricing, and risk
- AI-generated explanations for why products are promising
- Ranked product recommendations
- Web-based dashboard for exploration and filtering

## Tech Stack
- Backend: Python, FastAPI
- Database: PostgreSQL
- AI: OpenAI API
- Frontend: Next.js, Tailwind CSS
- SaaS: Stripe, Auth, Cloud Deployment
